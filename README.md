# NGNGA

NGNGA 是一个基于 Flutter 开发的第三方 NGA 客户端。

## 限制

- 仅支持 Android 系统
- 只能使用 Cookie 登陆

## 安装

你可以在 GitHub 的 release 下载到预先编译好的 apk 文件；

也可以从源码[重新编译](https://github.com/PoiScript/NGNGA/wiki/%E5%BC%80%E5%8F%91#%E6%89%93%E5%8C%85)。
